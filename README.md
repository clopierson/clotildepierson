# Clotilde Pierson's Professional Website

## Contributing

### First Time

[Set Up Your Development Environment](https://www.gatsbyjs.com/docs/tutorial/part-0/)

Optionally fork then clone this repository:

```shell
  git clone https://github.com/clopierson/clotildepierson.git
```

Install dependencies:

```shell
   npm install
```

### Start Developing

```shell
  gatsby develop
```

Your site is now running at http://localhost:8000!
Edit `src/pages/index.js` to see your site update in real-time!
